# Goon Tracker

Track sessions in five categories: **Porn**, **Hentai**, **Dojin**, **Illustration**, **Eroga**.

Two circular streaks:

- **Goon Streak** — consecutive days with at least one log
- **Dry Streak** — consecutive days without a log

Data is stored locally in your browser.

## Dev

```bash
npm install
npm run dev
```
